# 📘 Assignment: Python Advanced

## 🎯 Objective

Strengthen your Python problem-solving skills by working with advanced concepts like list comprehensions, dictionaries, and reusable functions. You will build a small data-processing program that is clean, efficient, and easy to extend.

## 📝 Tasks

### 🛠️ Data Processing with Comprehensions

#### Description
Create a program that takes a list of student scores and transforms it into useful summaries using list and dictionary comprehensions.

#### Requirements
Completed program should:

- Store at least 10 numeric scores in a list.
- Use a list comprehension to create a new list of passing scores (70 and above).
- Use a dictionary comprehension to map each score to a label (`"Pass"` or `"Needs Improvement"`).

### 🛠️ Reusable Analytics Functions

#### Description
Write reusable functions that analyze the score data and print a short report for the user.

#### Requirements
Completed program should:

- Implement a function that returns the average score.
- Implement a function that returns the highest and lowest scores.
- Print a clear summary report showing average, highest, lowest, and total passing students.
